ChatGPT Export Extractor

Split your ChatGPT export into one text file per conversation.

Local-only. Your data never leaves your browser — no upload, no server, no analytics. This page is a single static file you can save and run offline (file://).
How to get your ChatGPT export
  1. In ChatGPT, go to Settings → Data controls → Export data.
  2. You'll get an email with a download link. Unzip it.
  3. Inside you'll find either conversations.json or — for large accounts — a set of conversations-000.json, conversations-001.json, … files.
  4. Drop them on the box below.
